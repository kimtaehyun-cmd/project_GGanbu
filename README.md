# 캐릭터 챗봇 서비스 (GGANBU)

### 👨‍🏫프로젝트 소개
---
__GGANBU__ 는 사용자가 캐릭터의 성격, 말투, 이미지 등을 직접 커스터마이징, 생성하여 자유롭게 소통할 수 있는 서브컬쳐 챗봇 서비스 입니다.

### 🔧개발 환경
---
- Front : React
- Back : FastAPI
- 버전 및 이슈관리 : Github, Jira
- 협업 툴 : Notion, Jira, slack
- 서비스 배포 환경 : Jenkins
- 디자인 : Figma
